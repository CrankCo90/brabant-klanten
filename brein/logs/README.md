# Logs — dag- en run-rapporten

> Hier komen korte rapporten: wat is er per dag/run toegevoegd, gemaild, beantwoord.
> Handmatig nu; later schrijft de n8n-/auto-workflow hier automatisch Markdown-rapporten naartoe
> (bv. `2026-06-17-run.md` → "Gelderland +14, totaal Hondentrimsalons 154").

## Conventie
- Bestandsnaam: `JJJJ-MM-DD.md` (één per dag) of `JJJJ-MM-DD-<onderwerp>.md`.
- Begin met een kop `# <datum> — <korte titel>` en daaronder bullets.

## Bestaande dagverslagen
